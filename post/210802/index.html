2021-08-02 # 聚会 # 6+3 # 随笔

昨晚，原本打算三人到新近“正名”的红旗广场随便走上一走，散散心，却又意外地加入了 6+2 组合，于是，活动便丰富多了——几人轮换打羽毛球，共同教小郝学骑车，到老街闲走并在大排档聚餐，吃了把“夜宵”，各自到家，又是近十一时了。

或许这种放纵的生活并不符合我们三人一贯以来的作息与饮食习惯，但假期之中，偶尔放纵一次又如何，何况，能与情意如此相投的两家人建立这种融合了友情与亲情的亲密关系，大家一起抱团取暖，抱团取乐，老天让我们在如此年龄、如此境况之下给予我们如此眷顾、如此垂爱，那还有什么更多可顾虑的呢？！

有你们伴我们左右，真好！！！
