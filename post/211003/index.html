新“新生”

2021-10-03 # 心态 # 治疗 # 新生 # 不适 # 自规 # 随笔

近一个月来，胃部不适给我带来的困扰一直萦绕不断，历经针灸、中药、住院、胃镜乃至病理等数道关口的诊断与治疗，现在仍难以说有所好转，看来还需要较多时日的治疗与休养才可啊！

且不论这次的不适症状何时能够好转，单单从这次的经历（包括不适本身及相关检查带来的担忧）而言，就颇为值得我来深入思考一下对今后的生活方式、心态等多方面进行大力调整的必要性和紧迫性了，也许，这已经是老天给我的为数不多的自我调整的机会了，再不抓住就真的悔之晚矣了！

好了，直入正题吧，现就即日起将要进行的诸方面调整罗列如下，以便自我查验与对照——

  • 心态 ：禁强迫（心理+动作）、 常微笑、不抱怨、慢慢来、断舍离、小确幸、冥想+佛经……

  • 饮食：清淡（适当蔬果）、七分饱、忌烧烤……

  • 药物：少抗生素（强力药）、适度中药+理疗……

  • 运动 ：不久坐、常走动（早午走+哑铃等）……
