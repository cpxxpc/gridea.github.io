儿童节

2022-06-01 # 假日 # 疫情 # nn # 随笔

今天是六一儿童节，难得的nn得以基本上放松了一天，本次解封后终于与爷爷在我家共同吃了一顿午饭，又与姥姥共至鸟浪广场、西海岸浴场等处以车代步，游玩了一下午。说实话，对于西海岸附近繁密的车流、人流，我还是有些出乎意料的，由此可见，这一次长达二十多天的封锁，给周围的人们造成了多大的心理与胃口的影响，以至于，这处小小的人造海滩，竟然在这样一个儿童节的午后，为我们呈现出了如此热闹非凡的景象（至于残余疫情的隐患，在这里就姑且不论了）。

至于nn近期的学习与情绪状态，以后有机会再说吧，姑且让我们先轻松上这多半天吧！
